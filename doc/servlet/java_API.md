# API

## 用户

### 用户登录

- url:`localhost:8080/user/login`
- method:`POST`
- data:`username`,`passord`
- return:`status`,`userid`

示例

```json
// 发送数据
{
	"username": "pp",
	"password": "123456"
}

// 返回数据(成功)
{
	"status": "success",
    	"userid": 1
}

// 返回数据(失败)
{
	"status": "fail"
}
```



### 用户注册

- url:`localhost:8080/user/register`
- method:`POST`
- data:`username`,`passord`
- return:`status`

```json
// 发送数据
{
	"username": "pp",
	"password": "123456"
}

// 返回数据(成功)
{
	"status": "success"
}

// 返回数据(失败)
{
	"status": "fail"
}
```
